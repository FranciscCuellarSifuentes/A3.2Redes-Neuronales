Esta funcin load_and_process_image realiza un procesamiento completo de imgenes para prepararlas para un modelo de clasificacin de dgitos como MNIST. Primero carga la imagen en escala de grises y verifica su correcta carga. Luego aplica una serie de transformaciones: 1) umbralizacin para convertirla a blanco y negro (invertido con THRESH_BINARY_INV), 2) suavizado Gaussiano para reducir ruido, 3) mejora de contraste con CLAHE (Adaptive Histogram Equalization), y finalmente 4) redimensionamiento a 28x28 pxeles. La imagen resultante se normaliza (escalando valores entre 0 y 1) y se reformatea a las dimensiones esperadas por el modelo (1, 28, 28, 1). Este pipeline de procesamiento est optimizado para mejorar la calidad de imgenes de dgitos manuscritos, haciendo que sean ms legibles para el modelo y manteniendo compatibilidad con el formato MNIST.

import cv2

def load_and_process_image(image_path):
    """
    Carga y procesa una imagen para que sea compatible con el modelo MNIST.

    Parmetros:
    - image_path (str): Ruta de la imagen a cargar.

    Retorna:
    - processed_image (numpy array): Imagen procesada y redimensionada a 28x28.
    """
    image = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)

    if image is None:
        print(f"Error al cargar la imagen: {image_path}")
        return None

    _, thresh = cv2.threshold(image, 128, 255, cv2.THRESH_BINARY_INV)

    blurred = cv2.GaussianBlur(thresh, (5, 5), 0)

    clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
    contrast_enhanced = clahe.apply(blurred)

    resized = cv2.resize(contrast_enhanced, (28, 28))

    processed_image = resized.reshape(1, 28, 28, 1) * 255.0

    return processed_image

Para la mejora 1 se implement un suavizado inteligente con filtro Gaussiano; al aplicar este filtro, se elimino el ruido y las imperfecciones menores de las imgenes, preservando al mismo tiempo los bordes y trazos esenciales de los dgitos. Esto es til para que el modelo interprete correctamente escrituras con trazos irregulares o fluctuaciones en la presin del trazo.

La segunda mejora se hizo en la arquitectura de las capas descendente se implement una estructura profunda con cinco capas densas (256, 128, 64, 64, 32 neuronas), diseada para extraer patrones jerrquicos: las primeras capas capturan caractersticas generales de los dgitos, mientras que las posteriores afinan proesos en especfico. Esta progresin controlada ayuda a prevenir sobreajuste al reducir gradualmente la capacidad del modelo.

Falmente, para optimizar el contraste con CLAHE. La tcnica adaptativa mejora localmente el contraste, realzando detalles finos en zonas tanto oscuras como claras sin distorsionar reas bien iluminadas. Es particularmente efectiva para distinguir dgitos que se pareecen entre s comoel 5 y 6 o el 3 y 8, incluso cuando vara la iluminacin o calidad de la imagen original.
